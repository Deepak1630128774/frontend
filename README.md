# Energy Strategy Hub

## Overview

This frontend powers the Energy Strategy Hub experience for analytics, admin workflows, and multi-tenant authentication.

## Local development

Requirements:

- Node.js
- npm

Run locally:

```sh
npm install
npm run dev
```

Build for production:

```sh
npm run build
```

Run tests:

```sh
npm run test
```

## Stack

- Vite
- TypeScript
- React
- Tailwind CSS
- shadcn/ui
- Vitest
